## Key Access Commands
- Grafana: `kubectl port-forward svc/grafana-service -n monitoring 3000:3000` (admin:admin)
- Prometheus: `kubectl port-forward svc/prometheus-stack-kube-prom-prometheus -n monitoring 9090:9090`
- Consumer logs: `kubectl logs -n payment-system -l app=payment-event-consumer`
- Kafka events: see `docs/KAFKA.md` for consumer commands
- Load generator: `cd demo/loadgenerator && bash loadgen.sh multi-state`
- Build consumer: `make build-consumer && make docker-build-consumer && kind load docker-image payment-event-consumer:latest --name payment-demo && kubectl apply -k k8s/event-consumer/`
